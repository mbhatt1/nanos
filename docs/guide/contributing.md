All contributions must align with our [Project Charter](./charter.md):

1. Security - Never compromise security invariants
2. Simplicity - Keep code minimal and maintainable
3. Performance - Maintain efficiency

For detailed style and coding standards, refer to the CONTRIBUTING.md file in the repository root.
